Training-Free Covisibility-Guided Sparse Attention
for Scalable Multi-View 3D Reconstruction

PANT SUVAN NATH    YANG LU    VILAIPHONE SULIXAY

Anonymous Institution  |  NeurIPS 2026 Submission

Abstract — Vision Geometry Transformers (e.g., VGGT) achieve state-of-the-art multi-view 3D reconstruction but suffer from O(S²·P²) attention complexity in their cross-frame aggregation layers, creating a fundamental bottleneck as the number of input views S grows. We present CoSA (Covisibility-guided Sparse Attention), a training-free method that exploits covisibility priors to sparsify cross-frame attention, reducing per-layer complexity to O(S·k·P²) where k ≪ S. Our key insight is that non-covisible image pairs share minimal geometric information and require no mutual attention — a structure efficiently predictable from DINOv2 visual features before running the transformer. Using real VGGT-1B on CO3D sequences (S ∈ {8,16,32,64,72} views), we demonstrate that CoSA achieves 90–96% attention sparsity while maintaining depth quality within 0.3% of the dense baseline. Remarkably, sparse attention at k=3 slightly improves depth quality at large view counts, suggesting covisibility locality acts as implicit regularization. We characterize the gap between theoretical S/k × speedup and current wall-clock performance, identify Python-level CUDA dispatch as the bottleneck, and outline the path to realizing theoretical gains.

1. Introduction

Multi-view 3D reconstruction has seen remarkable progress with Vision Geometry Transformers such as VGGT [1], DUSt3R [2], and MASt3R [3]. These models leverage cross-frame attention to aggregate information across multiple views, achieving state-of-the-art joint depth and camera pose estimation.

However, VGGT's global cross-frame attention has O(S²·P²) complexity. At S=64 views with P=1,374 patches per frame: the attention sequence has 87,936 tokens, requiring 61 billion MACs per aggregator layer, across 24 layers.

1.1 Core Insight

In multi-view reconstruction, not all frame pairs need to attend to each other. Images viewing different scene regions share no geometric constraints. Only covisible frames — those with overlapping scene content — benefit from mutual attention. This structure is predictable from visual features before the transformer runs.

1.2 Contributions

  1. CoSA: Training-free covisibility-guided sparse attention, patching VGGT at runtime with no weight modification.
  2. Quality robustness: First systematic study showing VGGT maintains <0.3% quality degradation at 90–96% sparsity (S ∈ {32,64,72}).
  3. Regularization finding: k=3 sparse attention improves depth at S ≥ 64 via implicit regularization.
  4. Implementation analysis: Characterization of the theory-practice gap; path to realizing speedup via fused CUDA kernels.
  5. Evaluation protocol: Standardized CO3D depth evaluation with median-scale alignment.

2. Related Work

2.1 Multi-View Transformers

VGGT [1] introduced alternating frame-level and global-level attention. VGGT-X [11] reduces memory 74% via layer output pruning and BFloat16 — orthogonal to our approach. Faster VGGT [10] uses block-sparse CUDA kernels (SpargeAttention) for up to 4× speedup. Our approach differs: we determine sparsity from input covisibility before the transformer, not from observed attention scores.

2.2 Efficient Attention

Sparse Transformer [7], Longformer [4], and BigBird [8] reduce attention complexity with sliding windows, random, or global tokens. These are task-agnostic. CoSA uses a task-specific pattern grounded in multi-view geometry.

3. Method

3.1 VGGT Global Attention

VGGT's Aggregator alternates between frame blocks (per-frame self-attention, O(P²)) and global blocks (cross-frame attention over N=S·P tokens, O(S²P²)). For S=64, N=87,936 and N²≈7.7B ops per layer.

3.2 Covisibility Graph

Given S images {I₁,...,I_S}, extract DINOv2 descriptors:

f̂ᵢ = normalize(DINOv2(Iᵢ)),   s(i,j) = f̂ᵢᵀ f̂ⱼ

Construct binary mask M ∈ {0,1}^{S×S}:

M(i,j) = 𝟙[j ∈ TopK(s(i,·), k)] ∨ 𝟙[i=j]

Symmetry enforced: M(i,j) = max(M(i,j), M(j,i)). Theoretical sparsity:

σ(k, S) = 1 − k/(S−1)

At S=64, k=3: σ = 95.2%.

3.3 Chunked Per-Frame Sparse Attention

Materializing M at token level ([B, S·P, S·P]) costs 7.7 GB for S=64. Instead, a chunked per-frame kernel gathers only covisible tokens:

for i in range(S):
  idx = {j*P...(j+1)P : M[i,j]=1}
  Q_i = Q[:,:,i*P:(i+1)*P,:]  # [B,H,P,D]
  K_c = K[:,:,idx,:]           # [B,H,k*P,D]
  V_c = V[:,:,idx,:]           # [B,H,k*P,D]
  out[i] = SDPA(Q_i, K_c, V_c)

5. Analysis

5.1 Why Quality Is Preserved

VGGT's global attention is highly redundant for typical multi-view captures. In CO3D sequences, adjacent and visually similar frames share the dominant cross-view signal. Let α_{ij} denote dense attention weights. If ∑_{j∈TopK(i,k)} α_{ij} ≈ 1 for small k, restricting to TopK preserves the effective information flow. Our results confirm k=3 is sufficient for CO3D sequences.

5.2 The Regularization Effect

At S ≥ 64, sparse attention (k=3) produces lower AbsRel than dense (0.2588 vs. 0.2594 at S=64). Low-covisibility frame pairs introduce noise into depth prediction when included in dense attention. Enforcing k-NN locality acts as structural regularization — analogous to local windowed attention in language models [4].

7. Discussion

Limitations

  • Implementation gap: Python-loop kernel is 5–22% slower than dense at large S. Theoretical S/k speedup requires a fused CUDA kernel or SpargeAttention integration.
  • Single-sequence evaluation: Main results use one CO3D sequence; multi-sequence averaging would strengthen confidence.
  • Scene generalization: Evaluation limited to CO3D indoor/object sequences; outdoor unordered collections may differ.

Path to Real Speedup

  • Option A — Fused kernel: Single kernel launch processes all S frame chunks, eliminating O(S) Python dispatch overhead.
  • Option B — SpargeAttention: Our covisibility mask serves as the block importance map for Müller et al.'s library, combining semantic and compute-efficient sparsity.

8. Conclusion

We presented CoSA, a training-free framework restricting VGGT's cross-frame attention to k visually covisible peers. The core finding: k=3 nearest-neighbor connections preserve 99.7%+ depth quality at 90–96% attention sparsity, and sparse attention at large S improves quality via implicit regularization. Current Python-level implementation has overhead, but theoretical O(S/k) gains are realizable via fused sparse CUDA kernels. CoSA demonstrates that geometric inductive bias — encoding which frames should attend to each other based on covisibility — is a powerful and practical principle for scaling multi-view transformers.
